# PRESSURE

An interactive narrative game exploring themes of social pressure and self-doubt.

## Overview

PRESSURE is a personal game where you face your inner critic. Battle the voice of self-doubt through meaningful dialogue, using hand-drawn artwork and dynamic gameplay mechanics to break through psychological barriers.

## Features

- **Two-phase boss battle system** with dynamic difficulty
- **Custom collision detection** for card-launching mechanics
- **Progressive visual feedback** showing boss vulnerability
- **Haptic feedback** for tactile immersion
- **100% hand-drawn original artwork**
- **Authentic personal narrative** based on lived experiences

## Technical Stack

- **Swift** - Programming language
- **SpriteKit** - Game development framework
- **SwiftUI** - User interface framework
- **UIKit** - Haptic feedback integration

## Gameplay

1. **Face It** - Enter the tutorial
2. **Defeat FAILURE** - Launch response cards to counter attacks
3. **Break the Illusion** - Literally shatter the pressure to reveal freedom

### Phase 1: Generic Attacks
Face external pressures about success, career, and self-worth.

### Phase 2: Personal Attacks  
Confront intimate doubts rooted in real personal experiences.

## Design Philosophy

Every attack and response is drawn from authentic personal experience. The game isn't about defeating an enemy—it's about understanding that social pressure is fragile glass waiting to break.
